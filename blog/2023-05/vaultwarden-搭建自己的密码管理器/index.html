Vaultwarden 搭建自己的密码管理器
  • May 16, 2023
  • 1 minute read

安装

开源版本的地址:地址

最简单的就是使用docker镜像安装

docker pull vaultwarden/server:latest
docker run -d --name vaultwarden 
			-v /vw-data/:/data/ 
			-e WBE_VAULT_ENABLE=true 
			-e DOMAIN=https://yourdomain.com
			-p 10201:80 
			vaultwarden/server:latest

然后在Caddy中进行代理

yourdomain.com {
  tls xxxx@mail.com
  reverse_proxy / 127.0.0.1:10201
}

使用

输入地址后进入注册界面,使用邮箱进行注册

添加密码

浏览器使用时,可以使用Bitwarden的谷歌插件

移动端也可以使用Bitwarden的移动端

苹果可以在设置-密码-自动填充处就行选择
