SeaWeedFs使用

开源易用的分布式小文件存储系统

项目地址： github地址

SeaweedFS wiki:文档

启动

docker启动

docker run --name weed -d -p 9333:9333 -p 18082:8080 -p 18081:8080   -v /opt/weedfs/data:/data chrislusf/seaweedfs server -filer -dir="/data"

构建成功后，直接请求 domain:9333 可以看到管理界面

上传文件

1、直接将文件请求地址即可

2、先分配key,然后向key地址上传文件
